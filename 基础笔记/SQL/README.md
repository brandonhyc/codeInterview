## XML vs. JSON

|                       | XML                  | JSON         |
|-----------------------|----------------------|--------------|
| Verbosity/Complexity  | Harder               | Easier       |
| Validity              | DTD.. Widely used    | JSON Schema  |
| Programming Interface | Clunky               | Good         |
| Querying              | Strong. XPath XQuery | JSON Path... |
